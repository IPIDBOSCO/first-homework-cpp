# Contar valores positivos en un arreglo

En este ejercicio, se te proporcionará un arreglo de calificaciones, donde cada calificación es un número entero. 
Tu tarea es contar cuántas calificaciones son válidas, es decir, cuántas son mayores o iguales a cero.

# Instrucciones

1. Edita la función llamada `contarCalificacionesValidas` esta recibe un arreglo de enteros y su tamaño.
2. La función debe retornar el número de calificaciones válidas en el arreglo.
3. Prueba lo que realizando usando el test proporcionado

``bash
make test
```

Recuerde que no debe modificar el código del test, solo la función `contarCalificacionesValidas`, tampoco debe el nombre de la función.
